# このアプリについて

<span style={{ fontSize: '1.2rem', fontWeight: 'bold' }}>現在制作中です!!</span>

愛知工業大学の建物名や教室名は `G22120` が 10号館大講義室 であるといったように不規則であり、新入生や訪問者にとっては非常に分かりづらくなっています。
そのため施設や教室の場所を簡単に検索できるWebアプリを開発しました。

愛知工業大学 学生サービスグループの依頼により [システム工学研究会](https://www.sysken.net/) が開発・運用を行っています。

# 機能

## 施設確認

マップ上の施設を選択すると，その施設の情報を確認できます。

## 施設・教室検索

施設名や教室名で検索できます。

検索画面を開き，検索バーにキーワードを入力すると，該当する施設・教室が一覧表示されます。
一覧から選択すると，その施設・教室の場所へ移動します。

## 経路案内

施設を選択した状態で経路案内ボタンを押すと，現在地からその施設までの経路を表示します。
出発地は <span style={{ width: '1rem', height:'1rem', borderRadius: '50%', border: '2px solid var(--color-primary)', backgroundColor: 'white', display: 'inline-block', transform: 'translateY(2px)' }}></span> （青枠+白塗りの円），目的地は <span style={{ width: '1rem', height:'1rem', borderRadius: '50%', border: '2px solid var(--color-primary)', backgroundColor: 'var(--color-primary)', display: 'inline-block', transform: 'translateY(2px)' }}></span> （青塗りの円）で表示されます。

経路案内中は，画面下部に出発地と目的地が表示されます。
出発地をドラッグすると，出発地を変更できます。
目的地名を選択すると検索欄が開き，新たな目的地を選択すると変更できます。

## 現在位置確認

ブラウザの位置情報を許可すると，現在地を確認できます。
現在地は青い円で表示され，円の大きさは位置情報の精度を表しています。

画面右上の現在地ボタンを押すと，現在地へ移動します。

## 愛工大へフォーカス

画面右上の愛工大ボタンを押すと，愛知工業大学 八草キャンパスへ移動します。

## 方位角・ピッチ角変更

スマホやタブレットでは2本指で回転させることで方位角を変更できます。
3本指で上下にスワイプすることでピッチ角を変更できます。

PC では右クリックを押しながらマウスを動かすことで方位角とピッチ角を変更できます。

画面右上のリセットボタンを押すと，方位角とピッチ角をリセットします。

## ショートカットキー

PC 上で利用可能なショートカットキーを以下に示します。

- `<Meta> + M`: マップ画面へ移動
- `<Meta> + S`: 検索画面へ移動
- `<Meta> + H`: ヘルプ画面へ移動
- `<Meta> + O`: ボトムシートを開く
- `<Meta> + L`: 現在地をフォーカス
- `<Meta> + U`: 愛工大をフォーカス
- `<Meta> + P`: 経路検索
- `<Meta> + C`: 経路クリア
- `R`: 方位角・ピッチ角をリセット
- `Escape`: 各種閉じる

※ Meta は Windows では Ctrl キー，Mac では Command キーを指します。

## 開発者用

### 現在地変更

開発者用に現在地を変更する機能を実装しています。
クエリパラメータから移動軽度等を指定できます。

例: [/?lat=35.1832&lon=137.1114&acc=20](/?lat=35.1832&lon=137.1114&acc=20)

- `lat`: 緯度 (例: `35.1535`)
- `lon`: 経度 (例: `137.0479`)
- `acc`: 精度 (例: `10`)

### GeoJSON 作成画面

施設は GeoJSON 形式で管理しています。
GeoJSON の作成・編集のために [GeoJSON 作成画面](/geojson) を作成しました。

### 経路作成画面

経路はの作成・編集のために [経路作成画面](/route) を作成しました。

# 掲載情報

## 施設・教室

施設名，教室名およびフロアマップは 2025年度 学生便覧をもとに作成しています。
一部改変を加えている箇所や，独自に作成した箇所もあります。

誤りや登録漏れがある可能性があります。
発見された場合はご連絡いただけると助かります。[連絡先](#連絡先)

## クラブ

2026年1月時点の [愛知工業大学 団体一覧](https://www.ait.ac.jp/campuslife/club-activities/) をもとに作成しています。

あれば部室を，なければ活動場所を登録しています。
部室も活動場所もない場合や八草キャンパス外の場合は選択不可能としています。

誤りや登録漏れがある可能性があります。
発見された場合はご連絡いただけると助かります。[連絡先](#連絡先)

# Q&A

### 現在地が表示されません

ブラウザの位置情報の許可を確認してください。

### 教室等がヒットしません

便覧をもとに手動で登録しているため、登録漏れがある可能性があります。
発見された場合はご連絡いただけると助かります。[連絡先](#連絡先)

### 教室名等が誤っています

便覧や記憶を頼りに登録しているため、誤りがある可能性があります。
誤りを発見された場合はご連絡いただけると助かります。[連絡先](#連絡先)

### ペンギンが表示されました

ペンギンはかわいいです。特に意味はありません。
邪魔であればペンギンをタップすると消せます。

### シカがいます

はい，シカです。特に意味はありません。
右上の愛工大へフォーカスボタンを押すと大学に戻れます。
